# Weather Forecast Analysis Dashboard

## Overview
This project presents an interactive weather analytics dashboard built using Power BI.  
The dashboard integrates real-time data from a live Weather API to provide up-to-date weather insights.

## Key Features
- Real-time weather data integration via API
- Temperature, humidity, and air quality index (AQI) analysis
- Hourly and daily forecast visualizations
- Automated data refresh
- Location-based insights

## Tools & Technologies
- Power BI
- Weather API

## Dashboard File
- Weather_Forecast_Dashboard.pbix

## Notes
- This project focuses on data visualization and analytical dashboarding.
- API keys are securely managed and not exposed in this repository.
